StreamlineSimpleVTK
-------------------
Simple example using OSUFlowVTK, derived from original OSUFlow exmple

glStreamlineSimpleVTK
---------------------
Simple example using OSUFlowVTK with OpenGL, derived from original OSUFlow example
Press 's' to generate streamlines
Press 'd' to show result
Press 'a' to show animation

gcbStreamlineVTK
----------------
Simple example using OSUFlowVTK with OpenGL and GCB renderer, derived from original OSUFlow example
Press 's' to generate streamlines

StreamlineSimpleVTKPipeline
---------------------------
Simple example using vtkOSUFlow filter and VTK pipeline, derived from VTK example

StreamlineRenderVec
-------------------
A tool that computes streamlines interactively with vtkLineWidgets, using vtkOSUFlow filter and VTK pipeline.
Usage: StreamlineRenderVec <.vec file>

StreamlineRenderPlot3D
----------------------
A tool that computes streamlines interactively with vtkLineWidgets, using vtkOSUFlow filter and VTK pipeline.
Usage: StreamlineRenderPlot3D <q file.bin> <xyz file.bin>
